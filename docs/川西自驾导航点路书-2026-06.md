# 川西自驾导航点路书（2026-06）

研究时间：2026-06-07

适用行程：2026-06-13 到 2026-06-20，成都双流机场进出，四姑娘山、丹巴/中路藏寨、八美/塔公、姑弄村条件短停或过夜、康定/泸定下撤。

## 1. 结论先行

这份路书不要按“景点名一次导航到底”执行。川西很多导航坑来自三个问题：景点范围太大、平台把车带到村道/草地/非铺装路、酒店或景区内部不允许私家车直接到达。每天按 2-4 段设置导航点，先到确定的游客中心、停车场、镇区、酒店或民宿，再决定下一段。

总原则：

- 高德为主，百度/腾讯地图复核一次；两套地图差异很大时，以更成熟主路为准，不为了节省 10-20 分钟走乡道、小路或非铺装路。
- 每天出发前把当天所有点位按顺序加入收藏，并截图保存路线。
- 到山区民宿前，电话问店家“高德/百度到底搜哪个名字、车能否到门口、雨天是否要换停车点”。
- 天黑、下雨、大雾、头痛或司机疲劳时，不去姑弄村、甲根坝、野咖啡、理小路、鱼子西、八郎生都、格底拉姆。
- 小红书只用来识别坑，不用来追私藏机位。看到“导航某某小店/咖啡/窗口/秋千”的内容，只作为天气和路况很好时的候选，不作为主路线。

## 3. 分段导航路书

### 6/13：双流机场 -> 采购 -> 耿达/卧龙

推荐导航链：

1. `成都双流国际机场 / 租车点` -> `成都双流万达广场`
2. `成都双流万达广场` -> `映秀镇`
3. `映秀镇` -> `汶川归园居民宿` 或 `同福云轩` 或 `卧龙中华大熊猫苑神树坪基地`

执行要点：

- 双流万达作为采购锚点，优点是商业体、停车和餐饮更确定；如果取车点离沃尔玛英邦百货更近，也可以临时改，但评论里有“开车停车不如公共交通方便”的反馈，所以不把它作为默认。
- 映秀镇是进入山路前的缓冲点。到映秀后再看天色、雨势和司机状态，决定继续耿达还是卧龙。
- 如果 14:30 之后才拿到车，直接把终点降级到映秀/都江堰，不再赶卧龙。

不要直接导航：

- 不要第一天直接导航“四姑娘山镇”。当天有飞机、取车、出城和山路，晚上直接睡 3000m 附近不稳。
- 不要只搜“卧龙”。卧龙范围大，容易和实际酒店位置不一致。要搜具体酒店或神树坪基地作为锚点。

### 6/14：耿达/卧龙 -> 猫鼻梁 -> 双桥沟游客中心 -> 被梦叫醒

推荐导航链：

1. `汶川归园居民宿 / 同福云轩 / 卧龙神树坪` -> `猫鼻梁观景台`
2. `猫鼻梁观景台` -> `四姑娘山镇`
3. `四姑娘山镇` -> `双桥沟游客服务中心` 或 `四姑娘山景区-四姑娘山双桥沟停车场`
4. 到停车场后电话联系 `被梦叫醒酒店`，按景区和酒店接驳规则进沟

执行要点：

- 猫鼻梁只在天气好、能见度好、身体状态好时短停 20-40 分钟；风大、雾大、头痛就跳过。
- 双桥沟官方游客中心地址在四姑娘山镇附近，官方页面提供游客中心和咨询/救援电话；官方 2026-04-01 起双桥沟入园时间为 8:00-15:00。6/14 要尽量 14:30 前完成停车和入园流程。
- 被梦叫醒在双桥沟景区内。导航上能搜到酒店不代表私家车能直接开到房间门口，停车、行李和接驳必须提前电话确认。

不要直接导航：

- 不要直接搜“四姑娘山”。四姑娘山包括三沟一山，范围太大。
- 不要直接搜“被梦叫醒酒店”就一路跟导航开。先到双桥沟游客服务中心/停车场，再按酒店接驳走。
- 不要搜“巴朗山垭口”追高海拔野点。猫鼻梁够用。

### 6/15：双桥沟 -> 小金 -> 丹巴/中路藏寨

推荐导航链：

1. `双桥沟游客服务中心 / 双桥沟停车场` -> `小金县美兴镇`
2. `小金县美兴镇` -> `丹巴县城`
3. `丹巴县城` -> `已订中路藏寨民宿名`
4. 如果民宿 POI 不稳定：先搜 `中路藏寨 1 号观景台` 或 `中路藏寨 2 号观景台`，到丹巴后再电话让民宿指路

执行要点：

- 小金县美兴镇只是中途锚点，不是必须停留；它的作用是让导航走主路，并给加油/厕所/补给留余地。
- 14:00 还没离开双桥沟，就不要把中路山上民宿设成硬终点。改住丹巴县城或小金更安全。
- 进中路前先联系民宿：车能不能到门口、是否要错车、是否有固定停车点、晚到是否有人接。

不要直接导航：

- 不要只搜“中路藏寨”后跟着进村道。要么搜具体民宿，要么先到丹巴县城再二次确认。
- 不要天黑后第一次上中路。村道窄、错车难，夜间体验会很差。

### 6/16 默认：中路藏寨休整

推荐导航链：

1. `已订中路民宿` -> `中路藏寨 1 号观景台`
2. `中路藏寨 1 号观景台` -> `中路藏寨 2 号观景台`
3. 需要补给时：`中路藏寨` -> `丹巴县城`
4. 日落前：`丹巴县城 / 观景台` -> `已订民宿`

执行要点：

- 这天能不开车就少开车。中路的价值是低海拔恢复、藏寨、碉楼、星空和民宿露台。
- 如果住处本身视野好，观景台可减量，不要为了每个点位来回上下山。

不要直接导航：

- 不要临时加 `甲居藏寨`、`顶果山酒店` 或高山景区。它们会把休整日变成山路日。
- 不要夜里开车找星空机位。只在民宿院子/露台拍。

### 6/16 条件升级：丹巴/中路 -> 八美 -> 塔公/姑弄村过夜

启用条件：6/15 睡得好、无明显高反，6/16 天气好，且已订可取消、有供氧/取暖/停车的塔公或姑弄村住宿。

推荐导航链：

1. `丹巴县城 / 中路民宿` -> `八美镇`
2. `八美镇` -> `塔公寺停车场` 或 `塔公镇`
3. `塔公镇` -> `姑弄村`
4. 住宿按实际选择：
   - 稳妥：`康定塔公草原曼达拉酒店`
   - 特别：`YAK沐雪庄·姑弄里富氧山居`
   - 中间档：`拉岗牧场`
   - 性价比：`康定塔公夏央林民宿`

执行要点：

- 八美只做补给和缓冲；墨石公园不作为默认深玩。
- 塔公先吃午饭再去草原/寺庙，不空腹在 3700m 左右久站。
- 姑弄村可直接作为粗导航锚点，但停车和机位不标准。小红书种草笔记里有“导航姑弄村、路边/村民收费停车”的经验，也有避雷笔记集中吐槽车多错车、草地露营混乱和收费体验。因此只按主路短停处理。
- `见山·半山咖啡`、`噶日岗玛`可以作为天气好时的机位备选，不作为默认导航点。车多、下雨、路窄或头痛时不去。

不要直接导航：

- 不要只搜“塔公草原”后把车开到草地边。优先 `塔公寺停车场`、`塔公镇` 或酒店。
- 不要追小红书私藏机位、草地、溪流边、非铺装支路。
- 不要叠加甲根坝、鱼子西、八郎生都、格底拉姆、理小路。

### 6/17 默认：丹巴/中路 -> 八美 -> 塔公 -> 姑弄村短停 -> 康定

推荐导航链：

1. `丹巴县城 / 中路民宿` -> `八美镇`
2. `八美镇` -> `塔公寺停车场` 或 `塔公镇`
3. 天气好、身体好：`塔公镇` -> `姑弄村`
4. `姑弄村 / 塔公镇` -> `康定情歌广场城际酒店`

执行要点：

- 这天不要把 `新都桥镇` 设为必经点。新都桥是中转基地，不是本次核心景点；设成必经点可能把你带去镇上绕一圈。
- 从塔公/姑弄村下康定时，地图可能给 S434/机场路或 G318/折多山相关走法。出发前用高德和百度对比路况；如果其中一条显示施工、拥堵或恶劣天气，按当天主路和交警/导航提示走。
- 甘孜州 2026-06-01 路况汇总显示 S434 康定部分路段有交通管制，6/19 前部分点位半幅单边交替放行；所以不要把 S434 当成“必然顺畅”的摄影路。

不要直接导航：

- 不要直接导航 `新都桥住宿` 来替代康定。新都桥约 3400m+，对第一次高原不适合作为恢复夜。
- 15:00 还没到塔公、下雨、起雾或头痛，就删掉姑弄村，直接康定。

### 6/17 条件升级后：塔公/姑弄村 -> 康定/泸定

推荐导航链：

1. `塔公/姑弄村酒店` -> `姑弄村` 或 `塔公寺停车场`
2. `姑弄村 / 塔公镇` -> `康定情歌广场城际酒店`
3. 如果高反明显：`康定情歌广场城际酒店` 不入住，继续导航 `泸定眉州酒店`

执行要点：

- 上午最多补拍 1-2 个点，10:30 左右开始下撤。
- 如果前一晚睡不好，早上直接下撤，不再补拍。
- 到康定后若仍头痛、胸闷、恶心，继续降到泸定比住康定更稳。

不要直接导航：

- 不要从姑弄村再加甲根坝/鱼子西。它会把高海拔停留拉长。
- 不要在塔公/姑弄村夜拍后第二天继续赶长线。

### 丹巴到康定/泸定低海拔下撤备选

适用场景：天气差、高反、疲劳、不想住康定或不想走八美/塔公。

推荐导航链：

1. `中路民宿 / 丹巴县城` -> `泸定眉州酒店`
2. 如果当天只到康定：`丹巴县城` -> `康定情歌广场城际酒店`

执行要点：

- 这条下撤线的关键是不要加入 `八美镇`、`塔公镇`、`姑弄村` 这些高海拔景观点。让地图生成丹巴到康定/泸定的低海拔大渡河谷方向路线。
- 小红书关于 S211/S217 丹巴到康定/泸定的反馈有“车少、隧道多、路况良好”，但官方 2026-06-01 路况汇总显示 S217 丹巴县原林邦隧道 K166 在 2026-06-01 至 2027-12-31 单向交替放行，所以要接受等车和临时管制。
- 如果导航给出明显绕行八美/塔公的方案，那就不是低海拔下撤线，需要重新选择路线。

### 6/18：康定 -> 泸定

推荐导航链：

1. `康定情歌广场城际酒店` -> `泸定眉州酒店`
2. 午后短逛：`泸定眉州酒店` -> `泸定桥景区停车场`
3. 返回：`泸定桥景区停车场` -> `泸定眉州酒店`

执行要点：

- 泸定桥短逛 30-60 分钟即可。优先用酒店停车或景区停车场，不在老城临时绕车位。
- 甘孜州 2026-06-01 路况汇总显示 G318 泸定白日坝彩虹桥至泸欣大桥西岸路段 2026-05-25 至 6-26 每日 22:00 至次日 6:00 有夜间管制，端午假期不管制。白天影响较小，但晚到要复核。

不要直接导航：

- 不要直接搜“泸定桥”作为停车点，容易进入老城找车位。
- 不要 6/18 再折返高海拔景点。

### 6/19-6/20：泸定 -> 成都 -> 双流机场

推荐导航链：

1. `泸定眉州酒店` -> `成都双流机场附近酒店 / 成都还车点`
2. 需要中途休息时，在高德实时选择 `雅安` 附近服务区或高速出口，不提前锁死偏离高速的餐馆
3. 6/20：`成都酒店` -> `租车还车点` -> `成都双流国际机场 T2`

执行要点：

- 6/19 是端午首日，建议 8:00 前离开泸定。
- 6/20 晚 22:00 飞机，不要把还车安排到最后 1 小时。至少预留 2 小时给还车、摆渡、行李和安检。

## 5. 出发前一天复核动作

1. 把本文件里的导航点按日期保存到高德收藏夹，命名为 `6/13-1 双流万达` 这种顺序。
